# TaskProcessor

Learn background task processing. Will target some of the usual distribution
systems and have simple interval based publishers and consumers.

## Systems to Integrate With
- [ ] RabbitMQ
- [ ] Kafka
- [ ] Amazon SQS

Other pubsub systems?
